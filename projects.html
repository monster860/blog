  • FastDMM2

    This is a map editor for Space Station 13 that runs within a web browser. It makes use of the GitHub REST API to fetch art assets, existing maps, and code from forks of Space Station 13 and to allow opening pull requests from within the browser.

    The source code can be found here

  • Thread Safe

    This puzzle game was the winner of the 11th UM-Flint Game Jam in April 2022 and involves a thread mechanic where you are anchored to a string that can get caught on other objects. I did the art, the physics, and the music/sfx myself.
